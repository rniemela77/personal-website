Project Title:

Somni - Personality Analysis App

The vision:

I was motivated to make a mobile-first full-stack app that uses AI to analyze a user's personality.

Requirements:

  • must feel responsive and intuitive
  • must have user authentication
  • must store user personality data as well as quiz data
  • must have (affordable) AI personality analysis
  • must be maintainable and scalable
Frontend:

I chose Vue.js 3 because I had experience with it and it seemed to cover all the bases. Vue-router for SPA-like navigation, Pinia for state management, Bootstrap for fast UI iteration.

Backend:

I found that Netlify Functions was a great way to get a simple backend working quickly.

Database & User Authentication:

Because I wanted to keep the project simple and with a serverless architecture, so I chose Firebase for data storage and user authentication.

Other Tools:

I used Jest for testing, TypeScript for type safety, Vite for bundling, Stripe for payment processing, and OpenAI for AI analysis.

First Steps: Backend Development

I focused on getting the backend working first, so I created a simple Node.js backend using Netlify Functions that the frontend could call.

First backend functionality:
  1. Firebase user authentication
  2. Firebase `quizzes` collection to store quiz data
  3. Firebase `users` collection to store personality data
  4. Function to handle OpenAI API calls
  5. Stripe payment processing

Design: User Flow

As I worked on the backend, I began to think about the user flow and what the core pages would be.

Core pages:
  1. Dashboard - user views their entire personality profile
  2. Quiz - user takes a quiz
  3. AI Insights - user views the AI's analysis of their personality
User flow:
  1. User logs in
  2. User takes a quiz
  3. User views the quiz results
  4. (Repeat steps 2 and 3 until the personality profile has enough data for AI analysis)
  5. User views the AI's analysis of the user's personality

Development: Building the Frontend

As I settled on a user flow, I began the frontend development.

Building it out in Vue.js:

Creating the Vue.js components was a breeze.

I frequently asked myself (and AI) questions like:

  • Where does this logic go?
  • Does this belong in a component or a store?
  • What security concerns should I consider?
  • How will I feel coming back to this code in 6 months?

Final Product

The final product is a responsive and intuitive app that allows users to take personality assessments and read AI-generated insights about their personality profile.

The app is built to be maintainable and scalable. Adding new quizzes and AI analysis sections can be done with minimal effort.

In order to avoid high OpenAI API costs, I implemented a limit to the number of free personality analyses a user can take. I've also optimized the prompt to get the best results with the least amount of tokens.
